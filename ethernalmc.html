Brewery

Status · Ver0 ; Lowest Priority

This mod adds a long process of creating alcoholic Drinks.
With the right recipes, these Drinks will get special effects!
But if you drink too much, you will get drunk, aka negative effects!

CombatLog

Status · Ver0 ; Higher Priority

This mod kills a player upon them logging out of the Game while in combat.
This is done to prevent Combat Logging (Logging out while fighting to prevent fair Death)

CopperZ

DeadSouls

Status · Ver0 ; Higher Priority

This mod creates a Soul at the Place you Died upon Death. (Think Souls from Dark Souls)
If you fall into Lava, this will place the Soul on the Block you stood on before you fell into the Lava.
This can work for the Void too, except we turned it off to make room for the Void Totem.
